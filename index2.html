CSS e Flexbox:

Propriedade Função Exemplo
color Define a cor do texto. color: blue;
background-color Define a cor de fundo. background-color: yellow;
font-size Define o tamanho da letra. font-size: 20px;
margin Cria espaço externo ao redor do elemento. margin: 10px;
padding Cria espaço interno dentro do elemento. padding: 15px;
border Adiciona borda ao elemento. border: 1px solid black;
display: flex Ativa o Flexbox para organizar elementos em linha ou coluna. display: flex;
justify-content Alinha os itens horizontalmente. justify-content: center;
align-items Alinha os itens verticalmente. align-items: center;
flex-direction Define a direção dos itens. flex-direction: row;
gap Cria espaço entre os itens no Flexbox. gap: 20px;
