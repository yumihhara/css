Explicando CSS

CSS (Cascading Style Sheets, ou Folhas de Estilo em Cascata) é uma linguagem de estilo usada para definir a aparência visual de páginas web escritas em HTML ou XML. Enquanto o HTML estrutura o conteúdo, o CSS controla layout, cores, fontes, tamanhos e espaçamentos, permitindo separar a estrutura da apresentação para criar sites responsivos e atraentes.
